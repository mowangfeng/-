# 🐟 吃鱼游戏 (Fish Eater Game)

一款基于 Java Swing 开发的海洋生物吃鱼游戏。玩家操控小鱼，吞噬更小的鱼类成长，避免被大鱼吃掉！

## 🎮 游戏特性

- ✅ **WASD 键盘控制** - 灵活操控玩家小鱼
- ✅ **动态难度系统** - 难度随时间线性增长
- ✅ **进化机制** - 每次进化增长 20%，更换不同海洋生物
- ✅ **BOSS 鱼系统** - 随机出现高难度 BOSS
- ✅ **每分钟奖励波** - 每隔 60 秒刷新大量敌人
- ✅ **积分系统** - 吞噬鱼类累积积分
- ✅ **碰撞检测** - 准确的吞噬和死亡判定
- ✅ **游戏暂停** - 支持暂停/继续游戏
- ✅ **流畅动画** - 多线程处理，60FPS 稳定运行

## 🎯 游戏流程

1. **开始游戏** - 从一条小鱼开始
2. **吞噬成长** - 用 WASD 移动，吞噬比你小的鱼
3. **积分进化** - 积分达到阈值后体型增长 20%，更换海洋生物
4. **难度递增** - 敌人数量和速度逐渐增加
5. **避免死亡** - 不要被大鱼或 BOSS 鱼吃掉
6. **无限游玩** - 没有时间限制，持续挑战高分

## 🕹️ 控制方式

| 按键 | 功能 |
|------|------|
| **W** | 向上移动 |
| **A** | 向左移动 |
| **S** | 向下移动 |
| **D** | 向右移动 |
| **Space** | 暂停/继续 |
| **R** | 重新开始 |
| **ESC** | 退出游戏 |

## 📊 游戏数据

- **敌人增长** - 线性增长（每分钟增加更多敌人）
- **体型增长** - 每次进化增长 20%
- **奖励波周期** - 60 秒
- **积分阈值** - 每 100 分进化一次
- **BOSS 生成** - 随机出现，体型是玩家 2-3 倍

4. 开始玩游戏！

## 📖 游戏规则

1. **吞噬规则** - 只能吞噬比自己小的鱼
2. **死亡规则** - 被大鱼或 BOSS 鱼吞噬则游戏结束
3. **进化规则** - 每 100 积分进化一次，体型增长 20%
4. **难度递增** - 敌人数量和速度随时间增加
5. **无时间限制** - 持续玩下去，没有关卡限制

## 💡 游戏技巧

- 🎯 优先吞噬周围最小的鱼
- 🏃 遇到大鱼要立即逃跑
- 📍 利用地图边界躲避敌人
- ⏱️ 留意每分钟的敌人波浪
- 🆙 尽快进化，扩大吞噬范围
